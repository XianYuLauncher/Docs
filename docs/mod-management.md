# Mod 管理

## 安装加载器

要使用 Mod，需先安装加载器。请参考 [下载和安装 Minecraft](./download-minecraft.md) 中的「安装加载器」部分。

## 查找与下载 Mod

### 通过 Mod 下载

1. 在启动器资源下载页，点击顶部导航栏的「Mod」按钮
2. 在 Mod 页面浏览可用的 Mod
3. 可按类型、加载器、版本筛选
4. 点击 Mod 项查看详情
5. 选择要安装的版本，点击「下载」按钮
6. 选择要安装到的游戏版本
7. 等待下载和安装完成（如已开启「自动下载前置」功能，则会自动下载前置 Mod）

### 手动安装 Mod

1. 从第三方网站下载 Mod 文件（.jar 格式）
2. 在启动器版本管理页，将 Mod 文件拖放至启动器窗口并松开
3. Mod 会自动显示在 Mod 列表中

## 管理已安装的 Mod

1. 在启动器版本列表页，选择已安装 Mod 的游戏版本
2. 在版本管理中，找到「Mod」按钮
3. 可进行启用、禁用、查看详情、更新、转移、删除等操作
