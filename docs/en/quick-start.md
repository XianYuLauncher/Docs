# Quick Start

## Download the Launcher

1. Visit [XianYuLauncher Official Website](https://xianyulauncher.com/) or [Github](https://github.com/XianYuLauncher/XianYuLauncher/releases)
2. Click the "Download" button
3. Wait for the download to complete

## Install the Launcher

1. Locate the downloaded zip file
2. Extract the zip to a convenient location, e.g. `XianYuLauncher`
3. Follow the instructions in the zip file to complete installation
4. After installation, you can start using XianYuLauncher

## First Run

1. Double-click `XianYuLauncher` to start XianYuLauncher
2. Wait for the launcher to load
3. Read and accept the End User Agreement
4. Enter the launcher main interface
