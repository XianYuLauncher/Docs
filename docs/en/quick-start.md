# Quick Start

## Download the Launcher

1. Visit the official XianYuLauncher website
2. Click the "Download" button
3. Select the version suitable for your system (Windows 10/11)
4. Wait for the download to complete

## Install the Launcher

1. Locate the downloaded installer
2. Double-click to run the installer
3. Follow the installation wizard to complete the installation
4. After installation, a XianYuLauncher shortcut will appear on your desktop

## First Run

1. Double-click the desktop shortcut to launch XianYuLauncher
2. Wait for the launcher to load
3. Read and accept the user agreement
4. Enter the launcher main interface
