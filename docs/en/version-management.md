# Game Version Management

## View Version List

1. On the launcher's version list page, all installed versions will be displayed

## Edit Version

1. Select a version in the version list to open the version management page
2. On the version management page, you can manage the following version properties:
   - Game memory allocation
   - Java path
   - Game window settings
   - Mods, resource packs, shaders, etc.

## Delete Version

1. Select a version in the version list and right-click
2. On the version management page, click the "Delete" button
3. In the confirmation dialog, click the "OK" button
4. The version will be removed from the list
5. Note: Deleting a version will delete all related files, including saves and mods

## Import/Export Modpack

### Export Modpack

1. Select a version in the version list
2. Right-click the version, select "Export Modpack"
3. Select the content you want to export in the modpack
4. Click the "Export" button
5. Wait for the export to complete

### Import Modpack

1. In the version list area, click the "Import Modpack" button
2. Select the modpack file you want to import
3. Click the "Open" button
4. Wait for the import to complete
5. The imported modpack will appear in the version list
