# Launcher Settings

## Open Settings

1. On the launcher main interface, click the "Settings" icon in the sidebar
2. Enter the settings interface
3. You can see various settings options

## Basic Settings

### Personalization

1. In the settings interface, click the "Personalization" option
2. In the personalization settings, you can:
   - Switch between dark/light themes
   - Switch between Mica/MicaAlt/Acrylic/custom background themes

### Language Settings

1. In the settings interface, click the "Personalization" option
2. In the language settings, select the language you want to use
3. Supported languages include:
   - 中文（简体）
   - English
   - ...
4. The launcher will automatically switch to the selected language after restart

## Download Settings

### Download Source

1. In the settings interface, click the "Network and Download" option
2. In the download source settings, select the download source you want to use
3. Supported download sources include:
   - Official source
   - [BMCLAPI](https://bmclapi2.bangbang93.com/)
   - [MCIM](https://www.mcimirror.top/)

### Auto Download Dependency Mods

1. In the settings interface, click the "Network and Download" option
2. In the auto download dependency mods settings, you can:
   - Enable/disable auto download dependency mods

## Game Settings

### Java Settings

1. In the settings interface, click the "Java Settings" option
2. In the Java settings, you can:
   - Auto-find Java path
   - Manually specify Java path
